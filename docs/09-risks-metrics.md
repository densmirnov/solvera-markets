# Risks and metrics

## MVP risks
- Low solver liquidity.
- Offer spam.
- Winner sabotage.
- Sybil attacks.
- Verifier agent abuse.

## Mitigations
- TTL.
- Bond.
- Reputation.
- Fixed fee on expire.

## Success metrics
- Number of created intents.
- % of intents with successful `fulfill`.
- Average time to `fulfill`.
- Number of active solver agents.
